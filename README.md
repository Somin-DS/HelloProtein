| --- | --- |
| Add Page | - 사용자가 단백질 직접 추가
- 단백질 함유량 검색 기능
- 즐겨찾기 기능 |
| Stat Page | - 평균 단백질 섭취량 보여주기
- 차트로 보여주기(일/월 단위) |
| Setting Page | - 다국어기능(한글/영어)
- 목표 섭취량 설정(일반/운동용)
- 테마 설정(계란,근육,동그라미 등) |
